A Voting App

User Stories:

    authenticated, I can keep my polls and come back later to access them.
    2. authenticated, I can share my polls with my friends.
    3. authenticated, I can see the aggregate results of my polls.
    4. authenticated, I can delete polls that I decide I don't want anymore.
    5. authenticated, I can create a poll with any number of possible items.
    6. unauthenticated or authenticated, I can see and vote on everyone's polls.
  7. unauthenticated or authenticated, I can see the results of polls in chart form. (This could be implemented using Chart.js or Google Charts.)
  8. authenticated, if I don't like the options on a poll, I can create a new option.
